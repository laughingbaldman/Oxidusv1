# Knowledge Base

This folder contains the structured knowledge library for Oxidus.

## Core Areas
- raw_sources/ - unprocessed incoming material
- reading_queue/ - items awaiting classification
- drafts/ - partial or in-progress notes
- curated/ - validated, clean knowledge
- archives/ - deprecated or superseded content
- classification/ - taxonomy, topic types, and tags
- metadata/ - registries, manifests, and indexes

## Supporting Areas
- summaries/ - curated summaries
- glossary/ - key terms and definitions
- bibliography/ - sources and citations
- comparative/ - contrasts between topics
- case_studies/ - applied examples
- timelines/ - chronology and sequences
- people/ - biographies and thinkers
- frameworks/ - models and theories
- curricula/ - structured learning paths
- questions/ - inquiry sets
- experiments/ - tests and hypotheses
- cross_cutting/ - multi-domain themes
- domains/ - top-level subject areas
