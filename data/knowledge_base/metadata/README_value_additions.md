# Value Additions

This folder contains structural enhancements that improve retrieval, deduplication, and learning paths.

## Files
- ingest_rules.yaml: naming, metadata, and extension policies
- topic_synonyms.json: alias map for search normalization
- topic_to_domain.json: fast lookup from topic to domain
- dedupe_registry.json: hash registry to prevent duplicates
- curriculum_index.json: topic sequencing for learning paths
- graph_edges.json: lightweight topic relationship graph
- search_weights.json: retrieval scoring weights
